# 🚀 Data Structures & Algorithms Portfolio

Welcome to my Data Structures and Algorithms (DSA) repository! 

This repository is a comprehensive collection of the algorithms, data structures, and algorithmic paradigms I have implemented and analyzed. Rather than just storing raw code, I built this space to document my learning process, showcase my ability to analyze time/space complexities, and visualize algorithmic performance.

## 📂 Repository Structure

To make navigation easy, this repository is organized into distinct folders for each algorithm or weekly module. 

Inside each subfolder, you will typically find:
* **Source Code:** Clean, documented implementations (primarily in Python).
* **Performance Analysis:** A dedicated `README.md` breaking down the theoretical Time and Space Complexities (Big O, Theta, Omega).
* **Data Visualizations:** Graphs and plots generated via Matplotlib to benchmark the real-world execution time of the algorithms against theoretical expectations.

## 🛠️ Technologies & Tools
* **Languages:** Python (C/C++/Java may occasionally appear)
* **Libraries:** `matplotlib`, `time`, `math`, (will add others if used)
* **Concepts Covered:** Recursion, Divide & Conquer, Dynamic Programming, Sorting/Searching, Graph Algorithms, and Recurrence Relations (Master Theorem & Recursion Trees).

## 💡 Why This Repo Exists
I created this repository to bridge the gap between algorithmic theory and practical implementation. By benchmarking my own code and plotting the results, I can tangibly prove why a $\Theta(\log n)$ approach (like matrix exponentiation) crushes an $\Omega(2^{n/2})$ approach (like naive recursion) in the real world.

Feel free to explore the folders to see my code and read my interpretations of the data!
